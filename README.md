BemTV é uma aplicação web inspirada nos serviços de streaming modernos, desenvolvida para fins educacionais utilizando HTML, CSS e JavaScript puro, consumindo dados da API OMDb (Open Movie Database) para exibir informações sobre filmes e séries.
FUNCIONALIDADES
Pesquisa de filmes e séries em tempo real
Exibição de pôsteres dos títulos
Visualização de informações detalhadas:
Título
Ano de lançamento
Gênero
Classificação
Sinopse
Elenco
Diretor
Avaliação IMDb
Layout responsivo para diferentes dispositivos
Interface inspirada em plataformas de streaming
